## 15 Software Installieren 

### Debian / Ubuntu 

```
# apt ist der Paketmanager 
apt search apache2 
# seitenweise ausgabe
apt search apache2 | less 
# installieren 
apt install apache2 
```

```
apt install 
... installiert die Pakete (.deb - Dateien) und löst Abhängigkeiten automatisch auf
```
